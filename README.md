## Menjalankan Secara Lokal

1. Clone repository ini.
2. Buka folder project.
3. Jalankan dengan Live Server dari VS Code, atau buka index.html langsung di browser.

## Deploy (Netlify)

Untuk project ini (static site), Netlify adalah opsi paling cepat dan stabil.

### Langkah Deploy Netlify

1. Push project ke GitHub.
2. Login ke Netlify.
3. Klik Add new site > Import an existing project.
4. Pilih repository ini.
5. Build command: kosongkan.
6. Publish directory: `.`
7. Deploy.

### Konfigurasi Netlify yang Dipakai

- Build command: none
- Publish directory: .
